Nombre y Apellidos

APP EXAMEN Gestion Empleados API examen.tu_dominio.com/api/empleados





Tareas

Tarea 0. Lee todo lo que se pide y comprende la dimensión del ejercicio. Puedes crear en local la web con sus dos subcarpetas y luego subirlo todo el hosting para hacer las mínimas modificaciones. O puedes ir subiendo poco a poco y hacer cambios en el hosting.

Tarea 1. Subir esta web a tu hosting y cambiar nombre y apellidos. La URL de acceso debe ser examen.tu_dominio.com y la página debe llamarse examen.html

Tarea 2. Subir la app sobre gestion de empleados a una subcarpeta llamada app dentro de la web del examen Crea un subdominio llamado appexamen.tu_dominio.com y enlázalo con la app. Enlaza el botón de la web correctamente. Haz que la app se abra automáticamente en otra pagina.

Tarea 3. Subir la api sobre gestion de empleados a una subcarpeta llamada api dentro de la web del examen. Hacer accesible, por medio del .htaccess, la API con la URL del la web del examen, es decir, API -> examen.tu_dominio.com/api/empleados. Por úlimo, enlaza el botón de la web correctamente para que haga la consulta GET indicada. Haz que la app se abra automáticamente en otra pagina.

    A tener en cuenta

  • Tanto la app de empleados como la API usan y comparten la misma base de datos. Te proporciono el script para que la crees
  • En la API, para probarla, revisa la variable $uri, para ver como hace el explode.
  • OJO, con la herencia de .htaccess Vas a tener que poner otro archivo .htaccess en la subcarpeta de la app para que se abra automaticamente sin llamar a index.php

Entregas

Entrega1. Hoja de examen rellena con nombre y apellidos y URL de acceso a la web

Entrega2. En la tarea del Aula Virutal, adjunta una captura del contenido de la carpeta del examen, donde se vea el archivo examen.html y las dos carpetas, la de la app y la de la api.

Enrtega3. En la tarea del Aula Virutal, adjunta también toda la carpeta del examen comprimida en .zip
